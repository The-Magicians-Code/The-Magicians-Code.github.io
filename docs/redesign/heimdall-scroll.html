Heimdall · Vision pipeline for edge inference

Sight for an autonomous vessel

Bachelor’s thesis · TalTech, 2023

A real-time computer-vision system giving an autonomous Baltic Workboats Navy 18 WP patrol vessel the situational awareness to run its own sea trials. YOLOv5 re-architected through ONNX → TensorRT FP16 on a Jetson AGX Xavier — so the boat sees the sea from its own mast, without waiting on a signal from shore.

  • 3.5× faster inference
  • −26.8% energy
  • 99.8% GPU utilisation
